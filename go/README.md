# Go Examples

A collection of Go examples using the Otim Go SDK.

**SDK Version:** [`github.com/otimlabs/otim-go-sdk`](https://github.com/otimlabs/otim-go-sdk) 0.0.7

## Examples

### settle-ethereum-pyusd-to-base-usdc

Create a settlement accepting pyUSD tokens on Ethereum mainnet, settling to USDC on Base.

**Commands:**
- `go run main.go` - Create a new settlement orchestration
